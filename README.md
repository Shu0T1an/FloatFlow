# FloatFlow

FloatFlow is a lightweight desktop productivity assistant built with `Tauri 2 + React + TypeScript`.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm test`
- `npm run tauri dev`

## Notes

- The current local environment uses `npm`.
- Frontend tooling is pinned to `vite@6.4.2`, `vitest@3.2.4`, and `jsdom@26.1.0` for Node 20.18 compatibility.
- On this Windows machine, Rust/Tauri checks require `rc.exe` from the Windows SDK to be available in `PATH` or assigned to the `RC` environment variable.
